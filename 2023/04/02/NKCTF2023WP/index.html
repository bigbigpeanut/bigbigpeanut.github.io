    BV1Ai4y1V7rg，这是b站上面的博主视频(最后出题人也说这题的出题思路就是来自这里的)

    然后呢明白了原理：

    主要是利用了将文字转为十六进制码，然后以RGB的格式写入图片中。

    也就是说，图片的每一个像素其实都是文字

    接下来我们只要将RGB转为文字就可以啦（这里作者还将难度降低，让Red为256，使得图片更红，看过视频的做题者一看就可以知道思路）

    编写脚本：

    from PIL import Image

def decode(im):
width, height = im.size
lst = [ ]
for y in range(height):
    for x in range(width):
        red,green,blue = im.getpixel((x,y))
        if(blue | green | red) == 0:
            break

        index = (green << 8) + blue
        lst.append(chr(index) )

return ''.join(lst)

if __name__=='__main__':

all_text= decode(Image.open("1.bmp","r"))
with open("out.text","w",encoding = "utf-8") as f:
    f.write(all_text)`
