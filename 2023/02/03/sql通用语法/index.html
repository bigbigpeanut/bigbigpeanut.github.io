    SQL通用语法

    ddl 定义数据库和数据内容

    dml 对数据库进行增加和删减

    dql 查询数据库

    dcl 控制权限和用户数量

    注意：大小写可以不分

    ddl-数据库基本操作：

    查询：

    查询所有数据库：

    SHOW DATABASES；

/* show databases

    查询当前数据库：

    SELECT DATABASE();

/*select database();

/*别忘记括号

    创建(语法)：

    CREATE DATABASE [IF NOT EXISTS] 数据库名 [DEFAULT CHARSET
    字符集] [COLLATE 排序规则]；

    create database [if not exists] 数据库名 [default charset 字符集] [collate 排序规则];

    （后面的字符集和规则可有可不有 系统默认）

    删除：

    DROP DATABASE [IF EXISTS]数据库名;

    drop database [if exists] 数据库名;

    使用：

    use 数据库名;

    例：

    create database if not exists bigbigpeanut default charset utf8mb4;

    /*utf之后解释 utf8的支持内容较小 只有4个字节 所以建议使用utf8mb4
