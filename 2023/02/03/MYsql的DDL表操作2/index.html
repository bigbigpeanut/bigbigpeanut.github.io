    ddl-数据库基本操作：

    删除：

    DROP DATABASE [IF EXISTS]数据库名;

    drop database [if exists] 数据库名;

    使用：

    use 数据库名;

    ————————————————————————————分割线

    dl-表操作-数据类型：

    数值类型 字符串类型 日期时间类型

    数值类型：

    不多说了 和其他语言差不多 就是名字有些不同

    多了tinyint smallint mediumint

    然后就是int bigint float bouble

    decimal 依赖于M（精度）和D（标度）的值

    精度 标度：位数和小数位数

    例：
    age tinyint unsigned

    score double(4.1)

    字符串类型：

    有 ：char varchar 定长和变长字符串（占用空间的固定和可变，第二个比较智能 但第一个性能好）

    tinyblob tinytext

    blob text

    mediumblob mediumtext

    longblob longtext 这几种

    /*blob的意思和二进制有关 代表二进制数据的意思

    日期类型：

    有： date time year datatime timestamp这几种

    总结例子：

    create table emp(
   id int comment  '编号'， 
   worknumber varchar(10) comment  '员工工号',
   name varchar(10) comment  '员工姓名',
   gender char(1)  '性别',
   age tinyint unsigned  comment '年龄',
   idcardnumber char(18) comment '身份证号',
   entrytime date comment '入职时间'
)comment '员工表'
