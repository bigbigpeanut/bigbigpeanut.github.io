    MYsql数据库

    什么是数据库

    数据库：

    存储数据的仓库，数据是有组织的进行存储（和以前学的access是一类的）
    DataBase(DB)

    数据库管理系统：

    操纵和管理数据库的大型软件 DataBas Management System(DBMS)

    SQL：

    操作关系型数据库的编程语言，丁一岚一套操作关系型数据库统一标准（Structured Query Language）

    MySQL数据库

    MySQL官方提供了两种不同的版本:

    社区版(MySQLCommunity Server)免费，MySOL不提供任何技术支持

    商业版(MySQLEnterprise Edition)收费，可以试用30天，官方提供技术支持

    下载地址：

    https://dev.mysql.com/downloads/windows/installer/8.0.html

    这个是win系统的下载版本 也可以直接网页搜索mysql进行查询下载

    我们打开命令行输入

    net start mysal80
net stop mysql80

    然后打开mysql自带的客户端命令行

    mysql [-h 127.0.0.1] [-P 3306] -u root -p

    安装完成后咱们就可以开始学习啦
