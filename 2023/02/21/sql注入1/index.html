    1、寻找闭合
    可以用单引号直接闭合掉前面的语句

    -1’ union select database(),1#

    单行的注释：
    井号和–
    –+（在php中+是空格）

    特点：只能执行一句语句，可以使用union的语句联合起来

    union的限制：

    前后语句的返回参数数量要相同

    类型要相似
