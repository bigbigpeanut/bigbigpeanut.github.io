    dql-条件查询

    1、语法

    select 字段列表 from 表名 where 条件列表;

    例:

    select * from emp where gender='女' and age<25;

    2、条件

    看图（与c php大部分一样）

    例：

    between and 这种前后顺序不能换

    like ‘__’ 下划线代表几个

    like’%x’ %这种是类似linux 的*模糊匹配

    这种没见过 多注意

    来一两个例子：

    注意第九个的第二种写法 这也是可以的

    select * from 表名 where age in(条件,条件,条件);
