    DQL-介绍

    DQL英文全称是Data Query Language(数据查询语言)

    数据查询语言，用来查询数据库中表的记录。

    DQL在mysql等各种数据库中的使用频率是最高的，因为建表和删表大多数是开发人员或者管理人员进行的，而在互联网开放的现实，查询的人数量体量更为庞大，使用查询功能的人也是绝大多数的。

    所以，DQL是DDL、DML、DQL、DCL中使用频率最高的操作

    查询关键字：select

    dql-基本查询

    • 1、查询多个字段

      select 字段1,字段2,字段3,…from 表名;

      select from 表名;（通配符 尽量少用）

    • 2、设置别名

      select 字段1 [as 别名1],字段2[as 别名2]…from 表名;

    • 3、去除重复记录

      select distinct 字段列表 表名；
