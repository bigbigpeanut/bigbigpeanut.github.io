linux命令行最后全总总汇3

一、快捷键
ctrl+c 强制停止

ctrl+d 退出登出

history 查看历史命令

！命令前缀，自动匹配上一个命令

ctrl+r 搜索历史命令

ctrl+a|e 光标移动到命令开始或结束

ctrl+<-|-> 左右跳单词

ctrl+l 或clear命令 清屏

二、软件安装

1、命令行安装 yum命令

语法：

yum [-y] [install | remove |search] 软件名称

-y 自动确认 无需手动确认安装或卸载过程

install 安装

remove 卸载

search 搜索

注意：安装的限权比较高

安装需要联网（systemctl命令）

2、apt命令（小拓展）（ubuntu用的）

语法：

apt [-y] [install | remove | rearch]

三、systemctl命令

linux的大部分软件都可以使用systtemctl命令进行控制：启动、停止、开机自启动

语法：

systemctl [start | stop | status | enable | disable ] 服务名

分别是 启动 关闭 查看状态 开启开机自启 关闭开机自启

NetworkManager 主网络服务

network 副网络服务

firewalld 防火墙服务

sshd ssh服务

四、软连接（类似快捷方式）

1、ln(line)命令创建软连接

语法：

ln -s 参数1 参数2

-s 创建软连接

参数1 被链接的文件或文件夹

参数2 要链接去的目的地
