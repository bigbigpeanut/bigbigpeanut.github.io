linux命令行最后全总总汇

一、日期与时区

1、date命令

通过date’命令可以在命令行查看系统时间

语法：

date [-d] [+格式化字符串]

-d 按照给定的字符串显示日期 一般用于日期计算

%Y 年

%y 年份后两位数

%m 月份

%d 日

%H 小时

%M 分钟

%S 秒

%s 自1970-01-01 00:00：00 UTC到现在 的秒数

例：

date

date +%Y-%m-%d

date “+%Y-%m-%d %H：%M：%S”

补充：
-d 按照给定的字符串显示日期 一般用于日期计算

其中支持的时间标记为：
year年
month月
day天
hour小时
minute分钟
second秒

例：

date -d “+1 day” +%Y%m%d 显示后一天的日期

2、修改时区

获得root权限后

删除系统自带的localtime文件，并将/usr/share/zoneinfo/Asia/Shanghai文件链接为localtime

步骤为：

su - root

输入密码

rm -f /etc/localtime

ln -s /use/share/zoneinfo/Asia/Shanghai

ntp程序

ntp程序可以自动校准系统时间

yum -y install ntp

systemctl start ntpd

systemctl enable ntpd

也可以手动校准（需root）

ntpdate -u ntp.aliyun.com
