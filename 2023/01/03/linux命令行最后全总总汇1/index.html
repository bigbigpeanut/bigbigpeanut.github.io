linux命令行最后全总总汇1

1、 which命令

搜索命令的程序文件在哪

语法：

which 要查找的命令

2、find命令—按文件名查找文件

语法1：

find 起始路径 -name “被查找文件名”

最好获取root权限后再来，可以查找隐藏的文件

find可以和通配符搭配

例：

test* 以test开头的

*test 以test结尾的

test 含有test的

好像叫什么 模糊查询

语法2：

find 起始路径 -size +|-n[kMG]

    • 表示大于和小于

n表示大小数字

kMG表示大小单位

例：

查找小于10KB的文件：find / -size -10k

查找大于100MB的文件：find / -size +100M

查找大于1GB的文件:find / -size +1G

3、echo命令（比较常用，和管道符搭配使用，还不错）

可以使用echo命令在命令行内输出指定内容

语法：echo 输出内容

注意：内容复杂 含有括号的时候，可以加上””

4、反引号(`)

在esc下面

把包围的内容当成指令运行

例：

echo pwd

输出内容为：当前工作目录

5、重定向符（>>和>）

第一个是将左边命令的结果，覆盖写入到符号右边指定的文件

第二个是将左边命令的结果，追加写入到符号右边指定的文件

例：echo “hello linux” > text.txt

6、tail命令

使用此命令，可以查看文件尾部的内容，跟踪文件的最新修改

语法如下：

tail [-f -num] linux路径

-f，表示持续跟踪

-num，表示查看尾部多少行，不填默认10行

7、vi\vim文本编辑器（visual interface mix）

vim是vi的加强版 一般使用vim

总结：功能好用强大

部分语法：

vim 文件路径

如果文件不存在 那么此命令用用于编辑新文件

存在 那么编辑已有文件
