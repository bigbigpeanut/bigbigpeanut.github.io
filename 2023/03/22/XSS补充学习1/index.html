    这里总结一下三种不同：

    反射型发包 x=xiaodi => x.php =>回包

    存储型发包 x=xiaodi => x.php =>写到数据库某个表 => x.php =>回显

    dom型发包 x=xiaodi =>本地浏览器静态前端代码 x.php

    我们可以通过DOM-xss获取到他人前段的一些数据，比如登入的cookie，个人信息之类的

    不过好像其他类型也可以呀^

    三个的区别：目的、用法、前提不同
